Page-by-page review
Landing Page (Home) Very Good
Hero copy is clear and human — "Keep your AI safe and honest. One line of code." is excellent. No jargon.
Threat explanations are brilliant — you paired each technical term with plain English: "Prompt Injection → Someone tricks your AI". This is exactly right.
3-step "How it works" (pip install → wrap → protected) is simple and convincing.
Compliance section (GDPR, DPDP 2023, RBI, HIPAA) adds strong enterprise credibility.
CountUp animation for stats (99.9% threats caught) adds polish.
The live dashboard mockup shows hardcoded numbers (12,394 requests, 218 threats blocked). This is fine for now but could mislead if someone thinks it's real data.
"AI Guardrail · Open Source · Built in India" badge in hero — the Open Source claim needs to be real. If the pip package doesn't exist on PyPI yet, this is misleading.
CTA says "Protect My AI" → goes to /pricing. But there's no actual product to buy yet. The button should say "Join Waitlist" or "View Roadmap" to be honest.
Docs Page Good — impressive detail
3-panel layout (sidebar + content + TOC) is exactly what a real docs site looks like. Very professional structure.
15-agent decision mesh table is genuinely impressive — shows deep thought about the product architecture.
All 4 integrations covered: OpenAI, Anthropic, LangChain, Azure OpenAI — this is what real developers care about.
Copy buttons on code blocks, search bar in sidebar — these small UX details matter and show professionalism.
The quickstart says "pip install sentinel-ai-sdk" and references app.sentinel.ai — neither exist. This will confuse anyone who tries to follow it.
Docker + Kafka + Redis + Postgres self-hosted docs are very advanced. Great for showing depth, but may feel like overengineering for a project with no live backend.
Sidebar search doesn't actually filter — it's just a UI placeholder. Add a note "coming soon" or implement basic filtering.
TOC on the right always shows the same 5 items regardless of which doc is open. It should update per section.
Pricing Page Excellent
3-tier pricing (Free → Pro ₹4,500/mo → Enterprise Custom) is exactly the right structure for a SaaS.
Annual/Monthly toggle with "Save 20%" is a great touch — shows product maturity thinking.
FAQ section answers the most real questions (what's a request, what if I exceed limits, Indian language support).
Indian pricing in ₹ with DPDP/RBI compliance focus is a smart differentiator — nobody else is doing India-first AI security.
"Start Free Trial" on Pro plan creates an expectation of a working product. Should say "Join Beta" or "Get Early Access" until the backend is live.
hello@sentinel.ai email in Enterprise CTA — make sure this inbox actually exists and you'll respond, or change to a Google Form.
Sign In / Sign Up Good
Firebase auth with Google + Email/Password is production-grade. This is real, working infrastructure.
Friendly error messages (not raw Firebase codes) show attention to UX.
Clean design — glowing background, glass card, good form UX.
"Forgot password?" button has no handler — it's a dead button right now. Either implement it or remove it.
After signing in, user goes to /app (dashboard) — but the dashboard shows simulated/demo data. Make it very clear with a "DEMO" tag so users know.
App Dashboard Good — best page technically
Stat cards (requests, threats blocked, safe rate, latency) with trend indicators look genuinely professional.
Audit table with filter buttons (All, Block, Rewrite, Allow), badge colors, and decision scores is exactly what enterprise buyers want to see.
Live event stream panel with animated dots is a great UI detail.
"DEMO" tag on panels is honest and correct — keep this.
The "Gateway: Offline" status (amber warning) when no backend is connected is good. But the offline callout with pip install instructions could be clearer for non-developers.
Dashboard uses demo/simulated data. The most impactful next step for your project is wiring this to a real Python backend — even a tiny FastAPI endpoint.
Overall assessment
Design system quality: Professional. Space Grotesk + Inter, dark theme, glass cards, animations — this is better than most real SaaS landing pages.
Concept depth: Excellent. 15 agents, multi-tenant, FAISS, Bayesian consensus — shows you understand the problem space deeply.
India-first positioning (₹ pricing, DPDP 2023, Aadhaar PII, Hindi/Tamil support) is a genuine competitive angle.
The gap between what the site promises and what exists is the main risk. A developer who tries "pip install sentinel-ai-sdk" will hit a wall immediately.
One recommendation: publish a minimal real pip package that does even 20% of what's described. That single step turns this from a concept into a real open-source project.
