quickstart
from zero to protected in under 2 minutes
1
install the SDK
Works with any Python 3.9+ environment.
pip install sentinel-sdk
2
wrap your existing client
No refactoring. Works with OpenAI, Anthropic, or any LiteLLM-compatible client.
import openai, sentinel
# your existing setup
client = openai.OpenAI(api_key="sk-...")
# one line to protect it
safe = sentinel.wrap(
client,
policy="healthcare",
api_key="sntnl-..."
)
3
make calls — sentinel intercepts automatically
Identical API surface. Every call is screened in parallel.
response = safe.chat.completions.create(
model="gpt-4o",
messages=[{
"role": "user",
"content": user_input
}]
)
