# Changelog

## [1.0.0] - 2026-04-14

### Added
- Screen Selected Text command — scan any highlighted text through 19 security agents
- Trust Score command — get 0-100 trust assessment for any text
- Screen Entire File command — batch scan all string literals in current file
- Dashboard webview panel — gateway health, analytics, and agent mesh overview
- Gateway Health Check command — verify SENTINEL connection status
- Sidebar views: Gateway Status, Scan Results, Agent Mesh (19)
- Inline decorations showing threat scores (BLOCK/WARN/SAFE)
- Auto-scan on save (configurable)
- Right-click context menu integration
- Status bar indicator with live gateway status
- Full configuration via VS Code Settings
