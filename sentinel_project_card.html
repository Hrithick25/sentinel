What's new vs PHANTOM

Added a Hallucination Probe agent — uses NLI-based claim entailment (DeBERTa) to check whether every factual assertion in the LLM output is grounded in the retrieved context, returning a grounding score per claim. This is the #1 production blocker for enterprise RAG deployments and no open-source tool solves it end-to-end.

Built a Compliance Tagger that maps detected PII and sensitive content to specific regulatory obligations (HIPAA data categories, GDPR lawful bases, SOC2 controls) — and auto-generates a per-request compliance audit trail as a signed JSON attestation that can be exported for auditor review. Directly unlocks healthcare and fintech verticals.

Engineered a multi-tenant policy engine — organizations configure per-use-case safety profiles (stricter PII thresholds for HR bots, looser for code assistants) stored in Postgres, served at runtime via a TTL-cached policy resolver. Each tenant's config is version-controlled with rollback, enabling safe A/B testing of safety thresholds.

Packaged as a drop-in Python SDK + OpenAI-compatible proxy — `sentinel.wrap(openai_client)` or deploy as a sidecar; intercepts every API call transparently. This removes the biggest adoption friction: zero code refactor for existing applications. Also ships a LangChain callback handler and a LlamaIndex node postprocessor.

Designed a threat intelligence sharing layer — anonymized attack embeddings are (opt-in) aggregated across tenants into a federated FAISS index, improving detection for all users proportionally to network size. Customers who share data get a pricing discount — this is the product's network-effect moat.
